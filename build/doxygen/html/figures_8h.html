Detailed Description

Определение фигур для игры "Тетрис".

Каждая фигура представлена массивом 5x5 блоков (Block), где значение 1 обозначает занятый блок, а 0 — пустую ячейку. Эти массивы используются для отрисовки фигур на игровом поле и их управления.
