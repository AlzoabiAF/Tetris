Detailed Description

Файл для освобождения ресурсов в игре Tetris.

Этот файл содержит функции для освобождения памяти, выделенной для различных структур данных игры, таких как фигуры, игровое поле, информация об игре и игроках. Используется для предотвращения утечек памяти при завершении игры.
